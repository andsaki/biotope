# 実装計画: Issue #45 音のデフォルトをOFFにしたい

## ゴール
- 環境音の初期状態を OFF にする
- 初回クリック前に音が鳴らない/鳴れない状態でも UI 表示が矛盾しないようにする
- 既存のミュートトグル操作で ON/OFF を切り替えられること

## メモ
- 対象 Issue: #45
- 背景: ブラウザの自動再生制限により、初回クリックまでは AudioContext が起動しない
- 主な対象: `src/hooks/useAmbientSound.ts`, 必要に応じて `src/components/UI.tsx`

## 実装ステップ
1. 既存の環境音フックの初期状態と復帰処理を確認する
2. 初期値を OFF に変更し、ON 操作時のみ音量反映されるよう調整する
3. 型チェック・Lint・ビルドで回帰を確認する
