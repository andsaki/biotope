# 実装計画: 全季節×昼夜の環境音を実録寄りで揃える

## ゴール
- 春夏秋冬それぞれの昼夜に、雰囲気に合う実録環境音を割り当てる
- 季節/昼夜の切り替え時に実録サンプルを優先再生する
- ライセンス情報を README に明記する

## 対象
- `src/hooks/useAmbientSound.ts`
- `public/audio/ambient/`
- `public/audio/ambient/README.md`

## 実装ステップ
1. 各季節・昼夜に対応する公開ライセンス音源を選定する
2. サンプルファイルを追加し、読み込みマップを拡張する
3. 季節・昼夜ごとのサンプル選択ロジックを更新する
4. 型チェック・Lint・ビルドで回帰を確認する
